# FormAI — AI-Powered Form Builder

**Create intelligent forms in minutes. Analyze responses instantly.**

FormAI lets you describe what you need in plain language and generates a complete form automatically — fields, validation, conditional logic. No code required.

---

## Features

- **AI Form Generation** — Describe your use case, get a ready-to-use form in seconds
- **Smart Analytics** — Automatic response analysis, sentiment detection, and actionable insights
- **Conditional Logic** — Show or hide fields based on previous answers
- **Integrations** — Google Sheets, Notion, Slack, and webhooks
- **Auto Reports** — Daily summaries delivered to your email or Slack

## Tech Stack

- **Frontend** — Next.js 14, TypeScript
- **Database** — Supabase (PostgreSQL)
- **AI** — Claude (Anthropic)
- **Deployment** — Vercel

## Getting Started

```bash
npm install
npm run dev
```

Requires environment variables — see `.env.example`.
