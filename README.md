# FormAI 🚀
Constructor de formularios inteligentes con IA.

## LANZAR EN 30 MINUTOS ($0)

### PASO 1 — Supabase (ya lo tienes)
- Copia tu Project URL y Publishable key desde Settings → API Keys
- Ve a SQL Editor → pega supabase-schema.sql → Run

### PASO 2 — GitHub
```bash
git init
git add .
git commit -m "FormAI v1"
# Crea repo en github.com, luego:
git remote add origin https://github.com/TU_USUARIO/formai.git
git push -u origin main
```

### PASO 3 — Vercel (gratis)
1. vercel.com → New Project → importa tu repo
2. Agrega estas variables de entorno:
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
   - ANTHROPIC_API_KEY
3. Deploy
